# PowerShell Scripts Collection

This repository contains a collection of PowerShell scripts for various system administration tasks.

## Directory Structure

- **FileSystem-Management/** - Scripts for file system operations, permissions, and path management
- **[Other directories]** - Brief description of each

## Getting Started

Most scripts include detailed help information accessible via:

```powershell
Get-Help .\ScriptName.ps1 -Full
```

## Standards

All scripts follow the organization's PowerShell coding standards including:
- Standard header documentation
- Error handling
- Consistent color scheme for output
- Parameter validation
